Detailed Description

Ce fichier contiendra l'ensemble des fonction nécessaire au traitement des logement de la société

Author
VERDIER Nathan, MUZARD Thomas
Date
14 janvier 2022

Dans cette partit du programme, nous utilisons des tableaux pour exploité au mieux toute les notions vue en cours De plus, il était plus facile pour nous de faire les logement dans un tableau pour la simplicité du code

Ici, nous trions dans l'insertion du logement par numéro de logement
